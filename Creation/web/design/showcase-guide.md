# 组件演示入口（Storybook 同级）

## 入口说明
- 运行 `npm run dev` 后进入首页。
- 点击“创业路径”页签，再点击“查看演示弹窗”进入组件演示。

## 已覆盖状态
- Button：primary / ghost / accent
- Input：默认态、focus 态
- Tab：active / default
- Card：浅色与深色结构样式
- Modal：打开、关闭、遮罩点击关闭
- Navigation：激活态图标与普通态图标
- Table：头部与行样式
- Pagination：上一页/下一页
- Toast：触发、堆叠、自动消失

## 响应式断点
- 桌面：`>1200px`
- 平板：`<=1200px`
- 移动端：`<=980px`
